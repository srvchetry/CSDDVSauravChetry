About this Visualization

This project is a D3JS version of my Midterm Project. This narrative visualization is inspired by the Drill Down Narrative Visualization How This Bear Market Compares

This project includes 3 world maps each being a chloropleth showing emigration, remittances and GDP data for the countries. The legends using color palletes are used for understanding the map and drilling down on top countries in each map. Each map is interactive, meaning on clicking any country on any map would populate the interactive bar chart and a non-interactive line chart.

Conclusion: Remittances contribute to the GDP of the countries. Some countries( for e.g. Nepal ) have a third of their GDP from inward remittances. Emigration is a brain-drain phenomenon for the country, but that country also gets economic benefits from it.
