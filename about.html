About this Visualization

This project is a D3JS version of my Midterm Project. This narrative visualization is inspired from the Drill Down Narrative Visualization How This Bear Market Compares


Hybrid Structure: Drill Down Story

The narrative visualization loads the emigration 2017 chloropleth as the default.The user can move the mouse over the countries to find the emigration details about them. On clicking any country of interest, the bar chart and line chart showing the country's remittance details open up. On mouseover on the bar chart further displays details for the country for the chosen year. The other maps can be launched on clicking the buttons and they have the same onclick functionality.The drill down functionality is thus acheived with the combination of interactive chloropleth, mouse events, tooltip and interactive charts.

Scenes:

Interactive Chloropleth, Interactive bar chart and a Non Interactive line chart comprise all the scenes. The opening scene is a world map( chloropleth ) which is iteractive on mouse actions. The click event open the second scene which consists of the bar chart and the line chart

Annotations and Triggers:

The visualization opens with a message about what the visualization is and prompts the user to take action. The user can click on the visible buttons or hover over the map and click on any country to start the drill down. Each cloropleth have the color legend which helps understand the map colors. On mouseover, the color of the coutry changes to gray and also triggers a tooltip showing the details of the country. On clicing the country launches the bar chart and line chart for that country's data. On mouseover on the bar chart would again trigger the tooltip showing further details about the country

User Events and Parameters:

User can click on any buttons to launch a different map. User can move the mouse over the map, which will trigger the tooltip for the country. On mouseover on a country, the color of the country changes to gray and the tooltip displays the country name and value of that country based on which map the user is seeing. Use can click on any country on any map which then passes the country name to the bar and line charts. Based on the country name, the charts show data from a different file. For countries which do not have any data, the charts return empty.

Source Code:

Just right click on your browser and check my code. All libraries used are from d3js.org. All code is just HTML and D3JS
